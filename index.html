FastAPI Todo API

A modern, efficient API for managing your tasks

Python 3.11+ FastAPI MIT License

Features

  • Create tasks with priority levels
  • List and filter tasks
  • Find specific tasks by ID
  • Update task details
  • Delete tasks
  • Get task statistics
  • CORS enabled
  • Interactive documentation

Tech Stack

  • FastAPI - Modern web framework
  • Pydantic - Data validation
  • Uvicorn - ASGI server
  • Python 3.11+

Quick Start

Get your API running in minutes:

pip install fastapi uvicorn uvicorn main:app --reload

Your API will be available at http://localhost:8000

API Endpoints

GET / Welcome message and API info
POST /tasks Create a new task
GET /tasks List all tasks (with filters)
GET /tasks/{id} Get specific task by ID
PUT /tasks/{id} Update a task
DELETE /tasks/{id} Delete a task
GET /tasks/stats/summary Get task statistics
